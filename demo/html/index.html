Extender installation

Open the extender options by right clicking the extender icon and selecting options. Then change the setting Badge Entpoint to http://localhost/api/1.0/badge/

Supported research websites

  • PLOS: Partially supported, no filter (reason: technical problems with the PLOS JS framework)
  • Google Scholar: Partially supported, no big badges (reason: no detail page)
  • MS Academic Search: Fully supported
  • DOAJ: Fully supported
  • ScienceOpen: Fully supported
