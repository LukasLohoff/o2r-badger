Badger demo implementation

Extender installation

  • First, install the Chrome extension:
    • Open chrome://extensions/ in Chrome or select "More tools -> Extensions" in the menu ,
    • make sure that "Developer mode" in the top-right corner is enabled,
    • click "Load unpacked extension..."
    • and select the "extension" directory of o2r-extender (o2r-badger/demo/badger_demo/o2r-extender/extension) directory.
  • Then follow these instructions to configure the o2r-extender to run in a local environment:
    • Open the extender options by right clicking the new "o2r" extension icon and selecting options. Then change the setting Badge Entpoint to http://localhost/api/1.0/badge/
    • When visiting one of the supported websites make sure to allow mixed content in Chrome: Select "Load unsafe scripts" by clicking the shield icon in the address bar (detailed explanation)

Then visit one of the following research websites:

  • PLOS
  • Google Scholar
  • MS Academic Search
  • DOAJ
  • ScienceOpen

Cleanup

To stop the services and reset the repository to its previous state, press Ctrl-C or run docker-compose down. Then remove the badger_demo directory and the badger_demo.zip file.
